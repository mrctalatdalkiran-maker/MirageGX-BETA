# Mirage GX Browser

Mirage GX is a custom browser inspired by Opera GX.

## Features
- Tabs system
- Secret Window (Incognito)
- Panic Mode (P key)
- Built-in AI (Jarvis)
- Built-in Flappy Bird game

## Built With
- C#
- WinForms
- Microsoft WebView2

## Installation
1. Download release
2. Extract ZIP
3. Run Browser.exe
